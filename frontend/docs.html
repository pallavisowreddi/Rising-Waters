Data Preprocessing & Engineering

Raw datasets contain outliers and missing records that skew machine learning calculations. The preprocessing phase consists of:

  • Missing Values check: Handled via df.isnull().sum() to ensure integrity.
  • Outlier Treatment: Extreme rainfall anomalies capped at Interquartile Range bounds (IQR Method, bounds: Q1 - 1.5*IQR to Q3 + 1.5*IQR).
  • Feature Scaling: Data standardized using a fitted StandardScaler so all inputs share a mean close to 0 and a standard deviation of 1, stabilizing model margins.
